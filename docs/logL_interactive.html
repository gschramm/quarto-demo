Log-likelihood landscape for a 2-voxel problem

This page shows the Poisson log-likelihood \(\mathcal{L}(x_1, x_2)\) for the 2-voxel forward model

\[\bar{y}_1 = x_1 + b_1, \qquad \bar{y}_2 = x_1 + x_2 + b_2.\]

Use the controls below to set the measured counts \((y_1, y_2)\) and background levels \((b_1, b_2)\). The plot updates automatically.

  • Blue triangle — constrained maximum \(\hat{\mathbf{x}}\) (non-negative orthant)
  • Red star — unconstrained maximum
  • Black dots — 20 MLEM iterations starting from \(x_1 = x_2 = 2.5\)
